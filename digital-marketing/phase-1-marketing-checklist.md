# Phase 1 Marketing Checklist

Run **in parallel** with website go-live. Marketing priority: **Google Maps + reviews first**, everything else second.

---

## Week 1 — Foundation

### Google Business Profile
- [ ] Confirm listing claimed & verified
- [ ] Primary category: Dermatologist
- [ ] Phone, address, hours match `phase-1-website/config.js`
- [ ] Add website URL (Vercel or custom domain)
- [ ] Upload 10+ photos (clinic, doctor, equipment)
- [ ] Add all services with short descriptions
- [ ] Write business description (credentials + machines + location)
- [ ] Link Instagram in profile

### Reviews (start day 1)
- [ ] Copy **Get reviews** link from GBP dashboard
- [ ] Create reception QR code → print stand
- [ ] Add WhatsApp quick reply: “Google review” → link
- [ ] Train staff: ask every happy patient before they leave
- [ ] Respond to all existing reviews

### Website alignment
- [x] Real contact details in `config.js`
- [x] Google Maps link in `config.js`
- [x] Doctor photo + hero images in `assets/`
- [ ] Add live site URL to GBP
- [ ] Fix placeholder social links (`facebook`, `youtube` still `#` in config)
- [ ] Optional: footer “Review us on Google” button

### WhatsApp Business (clinic phone)
- [ ] WhatsApp Business app installed on clinic number
- [ ] Business profile: name, address, hours, website
- [ ] Quick replies: Book, Location, Pricing, Review link
- [ ] Away message outside hours

---

## Week 2 — Momentum

### Reviews target: **5+ new reviews**
- [ ] In-clinic ask: 100% of checkout patients (happy only)
- [ ] WhatsApp follow-up script sent within 2 hours (see `review-request-scripts.md`)
- [ ] Log: how many asked vs how many reviewed

### GBP activity
- [ ] 1 GBP post (tip + book CTA)
- [ ] Add 3 more photos if available
- [ ] Check Insights: calls, directions, website clicks

### Secondary listings (low effort)
- [ ] Practo clinic profile (optional `bookingUrl` in config)
- [ ] Justdial / Sulekha — same NAP as GBP
- [ ] Instagram bio: address, hours, WhatsApp link, Maps link

### Website trust
- [ ] Replace placeholder review cards with **real** snippets from Google (first name + area only, with permission)
- [ ] Submit site to Google Search Console (when domain stable)

---

## Week 3–4 — Baseline before Phase 2

### Reviews target: **15+ total**
- [ ] Review velocity: at least 2–3 new reviews per week
- [ ] Average rating maintained ≥ 4.8
- [ ] Negative reviews addressed within 48h

### Track acquisition
- [ ] Spreadsheet: source per new patient (Maps / referral / Instagram / other)
- [ ] WhatsApp enquiries per week (baseline number)
- [ ] GBP Insights saved monthly screenshot

### Light content (optional, 2–3 hrs/week)
- [ ] 2 Instagram posts: machine spotlight, doctor tip
- [ ] 1 Reel: 30 sec clinic tour or “what to expect” (no patient faces without consent)
- [ ] Do **not** let content delay review asks — reviews beat Reels for new clinics

---

## Phase 1 marketing “done” definition

You are ready to **scale** (Phase 2 ads, automation, CMS) when:

| Gate | Target |
|------|--------|
| Google reviews | 15+ |
| GBP completeness | 100% |
| Website live on GBP | 2+ weeks |
| WhatsApp conversations logged | 20+ |
| You know #1 patient source | Maps (validate with ask) |

Then read: `plans/phase-2-growth-system.md` + `content-calendar-template.md`

---

## What NOT to do in Phase 1

1. Spend on Meta/Google ads before 15 reviews
2. Buy fake reviews or review farms
3. Run two different phone numbers on different platforms
4. Post stock clinic photos — use real Dermacue photos only
5. Ignore WhatsApp response time — a Maps click dies if nobody replies
